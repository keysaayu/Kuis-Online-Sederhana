# Kuis-Online-Sederhana

# Deskripsi
Kuis online sederhana ialah website berbasis HTML, CSS, dan JavaScript.
Digunakan untuk mengerjakan soal kuis secara online tanpa backend dan database.

# Fitur
- Menampilkan soal
- User memilih jawaban
- Menghitung & menampilkan skor

# Alur-Logika
1.  User membuka halaman kuis
2.  User menjawab soal
3.  Javascript memeriksa jawaban
4.  Menghitung skor
5.  Hasil ditampilkan
